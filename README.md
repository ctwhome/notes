# notes
The first open source alternative to OneNote

Made with : 
- Fabricjs
- Vue 3 (Vite)
- Tailwindcss


Differences versus OneNote:

|    |      Notes      |  OneNote |
|----------|:-------------:|------:|
| Mouse Panning |  ✅    | ❌ |
| Drag to copy |   ✅    |   ❌ |
| Consistency across devices |   ✅    |   ❌ |
| All platforms |   ✅    |   ❌ |
| Animated Gifs |   ✅    |   ❌ |
| Videos |   ✅    |   ❌ |
| iFrames |   ✅    |   ❌ |
|  | |     |



### Development
```
clone reposotory
yarn install
yarn dev
```

*Build for production*
```
yarn build
```
